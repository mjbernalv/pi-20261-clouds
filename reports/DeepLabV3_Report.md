# DeepLabV3+ Neural Network for Cloud Image Segmentation

## Introduction

This project implements a DeepLabV3+ neural network for semantic segmentation of cloud images. Semantic segmentation is a computer vision task that assigns a class label to every pixel in an image. The objective of the model is to identify and segment four cloud categories: **Fish**, **Flower**, **Gravel**, and **Sugar**.

---

## Neural Network Architecture

The model is based on the **DeepLabV3+** architecture, a state-of-the-art semantic segmentation framework. DeepLabV3+ combines an encoder-decoder structure with Atrous Spatial Pyramid Pooling (ASPP) to effectively capture contextual information at multiple scales while maintaining detailed object boundaries.

### Key Components

### Encoder
The encoder extracts high-level features from the input image using convolutional layers. These layers progressively learn spatial and semantic information about cloud formations.

### Atrous Spatial Pyramid Pooling (ASPP)
The ASPP module applies multiple atrous (dilated) convolutions with different dilation rates. This allows the network to capture cloud structures of varying sizes without significantly increasing computational complexity.

### Decoder
The decoder refines the feature maps generated by the encoder and reconstructs detailed segmentation masks. This stage improves boundary localization and overall segmentation quality.

### Output Layer
The network produces four binary segmentation masks corresponding to the four cloud classes:

- Fish
- Flower
- Gravel
- Sugar

Input images are resized to **256 × 256 pixels** before being processed by the model.

---

## Data Preparation

The dataset contains cloud images and segmentation masks encoded using **Run-Length Encoding (RLE)**.

Before training, the masks were converted into binary images through a custom RLE decoding function. The preprocessing pipeline included:

1. Loading image files.
2. Decoding RLE masks.
3. Resizing images and masks to 256 × 256 pixels.
4. Normalizing image pixel values.
5. Organizing data into batches for efficient training.

A custom `SegmentationDataGenerator` was implemented to dynamically generate batches during training.

## Conclusion

The DeepLabV3+ neural network was successfully applied to cloud image segmentation. Training was performed using the Adam optimizer and Binary Cross-Entropy loss function for 15 epochs. Evaluation on unseen data demonstrated moderate segmentation performance, with strong recall but lower precision. Future improvements could include data augmentation, hyperparameter tuning, and alternative loss functions such as Dice Loss or Focal Loss.
